# climate-change-childhood-ari-ssa
Reproducible R-INLA analysis code for examining prenatal and postnatal climate anomalies, air pollution, vegetation, and childhood acute respiratory infections in sub-Saharan Africa through a DOHaD framework.


## Use of AI-Assisted Tools
AI-assisted tools were used to help refactor, organize, and document the original manuscript analysis scripts into a GitHub-friendly repository structure. The original statistical model specifications, exposure definitions, analytic decisions, interpretation of results, and manuscript conclusions were developed and reviewed by the authors. All code was reviewed and validated by the authors before repository release.
